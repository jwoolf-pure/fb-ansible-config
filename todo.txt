
install the following modules:
    ansible-vault
    netaddr
    paramiko  -- paramiko may already be installed but check to be sure.


edit the ansible.cfg file add the following line
library = ./local_modules

copy contents of local_modules into your project directory.

copy in the following roles:
    copy-template
    pure_cert
    pure_network
    pure_ir_passwd
    pure_ir_execline
